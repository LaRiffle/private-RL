# secret-breakout

## Run

```sh
python simple_breakout.py -h
usage: simple_breakout.py [-h] [--verbose VERBOSE] [--step_delay STEP_DELAY]
                          [--max_eps MAX_EPS] [--max_steps MAX_STEPS]
                          [--random_seed RANDOM_SEED]

Arguments for Simple Breakout

optional arguments:
  -h, --help            show this help message and exit
  --verbose VERBOSE     verbose logging
  --step_delay STEP_DELAY
                        delay after each step
  --max_eps MAX_EPS     Maximum number of episodes
  --max_steps MAX_STEPS
                        Maximum number of steps each episode
  --random_seed RANDOM_SEED
                        Random seed
```
